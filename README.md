# Harness Engineering

> **The discipline of building infrastructure that makes AI agents reliable in production.**

📖 **[Full Research Report (中文)](./REPORT.md)** | 🌐 **[Website](https://warren-wupeng.github.io/harness-engineering)**

---

## What is Harness Engineering?

```
Agent   = Model   (the trained neural network itself)
Harness = Tools + Knowledge + Observation + Action Interfaces + Permissions
```

> "The model is the horse. The harness channels its power. The engineer is the rider."

---

## The 7 Core Components

1. **Context Management** — What the agent sees and remembers
2. **Tool Orchestration** — How tools are accessed and executed
3. **Verification Loops** — Guardrails and quality checks
4. **Memory Persistence** — State across steps and sessions
5. **Observability** — Logging, metrics, cost control
6. **Safety & HITL** — Human-in-the-loop controls
7. **Architectural Patterns** — Loop design and agent coordination
